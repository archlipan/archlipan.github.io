#### Vue-cli 

辅助构建 Vue.js 代码的脚手架工具 : 目录结构, 本地调试, 代码部署, 热加载, 单元测试等

{% highlight javascript %} 

npm init  

npm install 

// install 脚手架
npm install -g vue-cli

// 脚手架可能会出现安装失败--> 找到对应的 vue-cli 安装目录,删除已有目录重新安装可解决

// 项目初始化 --> 选择对应的模版       
vue init <template-name> <project-name>     

npm run dev

       
{% endhighlight %}
